# Team Task Manager Frontend

A full-stack Team Task Manager frontend built with Angular and connected to a live Railway backend.

---

## Features

- User Login
- JWT Token Storage
- Dashboard
- Task Listing
- Angular Routing
- REST API Integration
- Railway Deployment

---

## Tech Stack

- Angular
- TypeScript
- CSS
- Railway

---

## Pages

### Login Page

- User authentication
- JWT token handling

### Dashboard

- View tasks
- Task status display

---

## Installation

### Clone Repository

```bash
git clone YOUR_GITHUB_FRONTEND_REPO
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
ng serve
```

Open:

```text
http://localhost:4200
```

---

## Build

```bash
ng build
```

---

## Deployment

Frontend deployed on Railway.
